# AI-Based Code Duplication Detector

Identifies near-duplicate code snippets that may not be detected by traditional methods due to minor variations.

## Free
```yaml
- uses: walshd1/ai-based-code-duplication-detector@v1
  with:
    gemini_api_key: ${{ secrets.GEMINI_API_KEY }}
```
